## 🔄 如何恢复英文界面

如果想恢复原始英文界面，有两种方法：

### **方法 1：删除备份文件**
```powershell
cd "E:\Program Files\claude-code-haha\src\components"

# 删除备份文件（会恢复为汉化后的版本）
Remove-Item ThemePicker.tsx.backup
Remove-Item Onboarding.tsx.backup
Remove-Item Config.tsx.backup
```

### **方法 2：手动恢复**
```powershell
cd "E:\Program Files\claude-code-haha\src\components"

# 用备份文件覆盖当前文件
Copy-Item ThemePicker.tsx.backup ThemePicker.tsx -Force
Copy-Item Onboarding.tsx.backup Onboarding.tsx -Force
Copy-Item Config.tsx.backup Config.tsx -Force
```

```python
def quicksort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quicksort(left) + middle + quicksort(right)

# 测试

## ⚠️ 注意事项

### **1. 更新会覆盖汉化**
- 下次运行 `bun install` 或更新项目时，汉化会被覆盖
- 解决方法：重新运行汉化脚本

### **2. 备份文件**
- 原始英文版已备份为 `.backup` 后缀
- 不要删除这些备份文件（除非你确定不需要恢复）

### **3. 汉化范围**
- 当前汉化了**最关键的 3 个文件**（23 处文本）
- 其他界面元素（错误提示、工具反馈等）仍然是英文
- 如需完整汉化，需要继续翻译更多文件

## 🛠️ 扩展汉化

如果想汉化更多内容，可以修改汉化脚本：

```typescript
// 编辑 apply-chinese-patch.ts
// 添加更多翻译规则到 replacements 数组

const replacements = [
    // ... 现有规则
    
    // 添加新规则
    { en: 'Loading...', zh: '加载中...' },
    { en: 'Success!', zh: '成功！' },
    { en: 'Error', zh: '错误' },
];
```

然后重新运行：
```powershell
C:\Users\Administrator\.bun\bin\bun.exe run apply-chinese-patch.ts
```
