## ⚙️ 配置说明

### **环境变量配置**

编辑 `.env` 文件：

```bash
# API 认证（必填）- 请替换为你的实际 API Key
ANTHROPIC_AUTH_TOKEN=your_token_here

# API 端点（已配置为 MiniMax）
ANTHROPIC_BASE_URL=https://api.minimaxi.com/anthropic

# 模型配置（MiniMax-M2.7-highspeed）
ANTHROPIC_MODEL=MiniMax-M2.7-highspeed
ANTHROPIC_DEFAULT_SONNET_MODEL=MiniMax-M2.7-highspeed
ANTHROPIC_DEFAULT_HAIKU_MODEL=MiniMax-M2.7-highspeed
ANTHROPIC_DEFAULT_OPUS_MODEL=MiniMax-M2.7-highspeed

# 超时设置（3000 秒）
API_TIMEOUT_MS=3000000

# 禁用遥测和非必要网络请求
DISABLE_TELEMETRY=1
CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC=1
```

## 🔧 常见问题解决

### **Q1: 如何重新进入欢迎界面？**

欢迎界面只在首次启动时显示。如需重新显示，请删除配置文件：

```powershell
# 删除系统配置
Remove-Item "$env:APPDATA\Claude" -Recurse -Force -ErrorAction SilentlyContinue
Remove-Item "$env:LOCALAPPDATA\Claude" -Recurse -Force -ErrorAction SilentlyContinue

# 重新启动
cd "E:\Program Files\claude-code-haha"
.\start.bat
```

### **Q2: 中文显示为乱码？**

如果看到类似 `馃殌 鍚姩...` 的乱码：

**解决方案：**

使用 `start.bat` 启动（已自动设置 UTF-8 编码）

或手动设置：

```powershell
chcp 65001
[Console]::OutputEncoding = [System.Text.Encoding]::UTF8
.\start.ps1
```

### **Q3: 如何切换主题？**

在设置界面中：

1. 按 `/` 打开命令面板
2. 输入 `theme`
3. 选择你喜欢的主题样式

可选主题：
- 深色模式
- 浅色模式
- 深色模式（色盲友好）
- 浅色模式（色盲友好）
- 深色模式（仅 ANSI 颜色）
- 浅色模式（仅 ANSI 颜色）

### **Q4: API Token 在哪配置？**

编辑项目根目录的 `.env` 文件：

```powershell
notepad .env
```

将 `ANTHROPIC_AUTH_TOKEN=your_token_here` 改为你的实际 Token。

## 🆘 故障排除

### **启动失败**

1. 检查 Bun 是否安装：`bun --version`
2. 检查 `.env` 文件是否存在
3. 检查端口是否被占用

### **无法连接 API**

1. 检查网络连接
2. 验证 API Token 是否正确
3. 确认 API 端点 URL 无误

### **中文显示问题**

确保使用 UTF-8 编码：
- Windows Terminal → 设置 → 外观 → 字体 → 使用支持中文的字体
- 或使用 `start.bat` 自动处理编码
